# Vi 명령어
* dan@ubuntu:~$ => 현재 나의 상태 : 환경 변수를 제공합니다
* env => 환경 변수를 보는 명령어
* echo => 반복하라는 명령어
* echo $USER => USER가 누구인지 알 수 있는 방법
* etc => 모든 파일을 모아놓은 디렉토리
* cat => 파일의 내용을 볼 수 있는 명령어
* PWD => Present Working Derectory
* ~ => Home 디렉토리를 의미
* $ => prompt (명령을 실행한다는 표현)
* $ => 일반적인 사용자 계정
* # => root  계정
* cd => change directory
* control L -> clear
* mkdir => 디렉토리 만들기
* vi 1.c => c 파일 하나 만들기
* vi 설정을 통해 개인화 할 수 있다 => 
* :w => 저장
* vi ~/.vimrc
* gcc 1.c => 파일을 돌려보는 방법
* which => 
* echo $PATH => path를 잡는 방법
* ./ 파일 이름 => 파일을 실행시키는 방법
* ex) a.out
* . => 현재 디렉토리
* ./a.out
* rm 삭제 하는 기능

* y(복사)
* d(잘라내기)
* p(붙여널기)
